# Congratulations!

You have successfully installed and started Obot using Docker.

## What You Learned

- How to run Obot using the official Docker image
- How to verify the container is running and check logs
- How to access the Obot UI

## Next Steps

- **Configure a model provider**: In the admin UI, add your API keys for OpenAI, Anthropic, or other providers.
- **Add MCP servers**: Browse the MCP catalog and enable servers for your agents.
- **Create an agent**: Build a custom agent with access to your MCP tools.
- **Explore the API**: The Obot REST API is available at `http://localhost:8080/api`.

For more details, visit the [Obot documentation](https://docs.obot.ai).
