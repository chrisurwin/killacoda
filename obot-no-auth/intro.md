# Getting Started with Obot

Obot is an open-source platform that provides everything an organization needs to implement MCP (Model Context Protocol) technologies. It enables you to:

- **Host MCP servers** for internal and external users
- **Set up MCP registries** for discovery and cataloging
- **Manage and monitor MCP usage** across your organization
- **Build agents and chatbots** that leverage MCP servers

In this scenario, you will install and run Obot locally using Docker, then explore the Obot UI.

## Prerequisites

- A valid API key from OpenAI, Anthropic, or another supported model provider

Docker is pre-installed in this environment.
